The \doxylink{classngsai_1_1DiPositionNormalizedKineticModel}{Di\+Position\+Normalized\+Kinetic\+Model} class is made to model the normalized IPD and PWD signal from Pac\+Bio CCS. Instead of modelling the raw kinetic signal like the \doxylink{classngsai_1_1DiPositionKineticModel}{Di\+Position\+Kinetic\+Model} class, this class models kinetic ratios (normalized). Each stretch of kinetic values (IPD or PWD) x1,x2,...,xL is normalized by the expected kinetic signal given the DNA sequence s1,s2,...,sL associated to the kinetic values. This model models a window of IPD/\+PWD signal of L consecutive bp. L is named the model size. This model is a special case of the \doxylink{classngsai_1_1PairWiseNormalizedKineticModel}{Pair\+Wise\+Normalized\+Kinetic\+Model}. The \doxylink{classngsai_1_1PairWiseNormalizedKineticModel}{Pair\+Wise\+Normalized\+Kinetic\+Model} accounted for all N = ((L$\ast$L) -\/ L) / 2 ~\newline
 pair-\/wise position interaction whereas this one only considers L-\/1 interactions between any two directly neighbouring positions in the window. Except for this, everything is the same as in \doxylink{classngsai_1_1PairWiseNormalizedKineticModel}{Pair\+Wise\+Normalized\+Kinetic\+Model}.

\doxysubsection{Detailed Description}
The \doxylink{classngsai_1_1DiPositionNormalizedKineticModel}{Di\+Position\+Normalized\+Kinetic\+Model} class is made to model the normalized IPD and PWD signal from Pac\+Bio CCS. Instead of modelling the raw kinetic signal like the \doxylink{classngsai_1_1DiPositionKineticModel}{Di\+Position\+Kinetic\+Model} class, this class models kinetic ratios (normalized). Each stretch of kinetic values (IPD or PWD) x1,x2,...,xL is normalized by the expected kinetic signal given the DNA sequence s1,s2,...,sL associated to the kinetic values. This model models a window of IPD/\+PWD signal of L consecutive bp. L is named the model size. This model is a special case of the \doxylink{classngsai_1_1PairWiseNormalizedKineticModel}{Pair\+Wise\+Normalized\+Kinetic\+Model}. The \doxylink{classngsai_1_1PairWiseNormalizedKineticModel}{Pair\+Wise\+Normalized\+Kinetic\+Model} accounted for all N = ((L$\ast$L) -\/ L) / 2 ~\newline
 pair-\/wise position interaction whereas this one only considers L-\/1 interactions between any two directly neighbouring positions in the window. Except for this, everything is the same as in \doxylink{classngsai_1_1PairWiseNormalizedKineticModel}{Pair\+Wise\+Normalized\+Kinetic\+Model}.

Inserts the normalized averaged IPD and PWD values in the model. First the IPD and PWD are normalized (IPDRs / PWDRs) using the model kmer map and then averaged (fw and rv). Each element of the averaged IPDRs / PWDRs corresponds to a position in the model. Note that the normalization process shortens the IPDRs / PWDRs signal stretches of the first K//2 and last K//2 values where K is the size of the kmer used for the normalization. Thus the kinetic values and sequence must have a length equal to S+2$\ast$K where S is the size of the model. ~\newline
