class \mbox{\hyperlink{classngsai_1_1ThreadPool}{ngsai\+::\+Thread\+Pool}}
\begin{DoxyCompactList}\small\item\em The \doxylink{classngsai_1_1ThreadPool}{Thread\+Pool} class implements a simple pool of working threads. At construction, $<$n$>$ threads are spawned and tries to get tasks to execute from a queue. Any access to the queue is synchonized through the use of a mutex. Jobs are added to the queue by calling add\+Job(task) where task is the result of std\+::bind(). The queue can be open -\/ in which case the jobs are effectively added to the queues -\/ or closed -\/ in which case nothing can be push into the queue anymore. Stopping the pool is done through calling \doxylink{classngsai_1_1ThreadPool_a2a8767845dcae378ce14975251bbb458}{join()} which will close the queue. \end{DoxyCompactList}\end{DoxyCompactItemize}
