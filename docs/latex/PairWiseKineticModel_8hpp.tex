class \mbox{\hyperlink{classngsai_1_1PairWiseKineticModel}{ngsai\+::\+Pair\+Wise\+Kinetic\+Model}}
\begin{DoxyCompactList}\small\item\em The \doxylink{classngsai_1_1PairWiseKineticModel}{Pair\+Wise\+Kinetic\+Model} class is made to model the raw IPD and PWD signal from Pac\+Bio CCS. This model models a window of IPD/\+PWD signal of L consecutive bp. L is named the model size. This model accounts for pair-\/wise relaitions ships between any two positions in the window. There are exactly N = ((L$\ast$L) -\/ L) / 2 pair-\/wise relations. The signal in the Lbp window is modelled using 2$\ast$N different distributions of signal -\/ N for IPD and N for PWD signal. Each distribution is modelled by a 2D histogram representing the signal at one position p in the window with respect to the signal found at the paired position q in the window where p != q. Overall the window signal is modelled by 2$\ast$N histograms h\+\_\+ipd\+\_\+p\+\_\+q and h\+\_\+pwd\+\_\+p\+\_\+q \+: h\+\_\+ipd\+\_\+1\+\_\+2, h\+\_\+ipd\+\_\+1\+\_\+3, ..., h\+\_\+ipd\+\_\+1\+\_\+L, h\+\_\+ipd\+\_\+2\+\_\+3, ..., h\+\_\+ipd\+\_\+2\+\_\+L, ..., h\+\_\+ipd\+\_\+\+L-\/1\+\_\+L and h\+\_\+pwd\+\_\+1\+\_\+2, h\+\_\+pwd\+\_\+1\+\_\+3, ..., h\+\_\+pwd\+\_\+1\+\_\+L, h\+\_\+pwd\+\_\+2\+\_\+3, ..., h\+\_\+pwd\+\_\+2\+\_\+L, ..., h\+\_\+pwd\+\_\+\+L-\/1\+\_\+L. The model can be trained (feed with data) in order to fill the underlying histograms and estimate the signal distributions at each pair of positions in the window. At this moment, the histograms are count histograms. Once the histograms have been filled, the model can be normalized in order to transform the 2$\ast$N histograms into 2$\ast$N probability densities. At this moment, the model can compute the likelihood of observing a a strech of signal of length L with IPDs X x1, x2, ..., xL and PWDs Y ~\newline
 y1, y2, ..., yL. The likelihood corresponds to \+: p(x1,x2 $\vert$ h\+\_\+ipd\+\_\+1\+\_\+2) $\ast$ p(x1,x3 $\vert$ h\+\_\+ipd\+\_\+1\+\_\+3) $\ast$... $\ast$ p(x1,xL $\vert$ h\+\_\+ipd\+\_\+1\+\_\+L) \end{DoxyCompactList}\end{DoxyCompactItemize}
