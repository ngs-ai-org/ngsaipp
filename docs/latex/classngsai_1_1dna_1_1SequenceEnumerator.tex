\doxysubsection{Detailed Description}
This class allows to generate all DNA sequences of a given length (kmer), in lexicographic order. The DNA base lexicographic order is defined in \doxylink{namespacengsai_1_1dna_ada1b7328bde664b8aa78a889c9371297}{ngsai\+::dna\+::base\+\_\+code} and in \doxylink{namespacengsai_1_1dna_a34d8261487b1304f0650d83af40c6682}{ngsai\+::dna\+::code\+\_\+base} in \doxylink{dna__utility_8hpp}{dna/dna\+\_\+utility.\+hpp}.
