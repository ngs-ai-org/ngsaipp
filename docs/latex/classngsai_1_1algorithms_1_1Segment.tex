\doxysubsection{Detailed Description}
The segment class is a data structure for the \doxylink{namespacengsai_1_1algorithms_a43c09d920ce5d1dcbbead35080eb8bc9}{find\+\_\+max\+\_\+scoring\+\_\+segments()} function implementing Ruzzo and Tompa 1999\textquotesingle{}s algorithm. The segment class allows to model a score sub-\/sequence inside a score super-\/sequence, together with the data required by Ruzzo and Tompa\textquotesingle{}s algorithm. A sub-\/sequence encompasses the elements contained in the range \mbox{[}from,to) of the super-\/sequence.
