Detailed Description

The segment class is a data structure for the find_max_scoring_segments() function implementing Ruzzo and Tompa 1999's algorithm. The segment class allows to model a score sub-sequence inside a score super-sequence, together with the data required by Ruzzo and Tompa's algorithm. A sub-sequence encompasses the elements contained in the range [from,to) of the super-sequence.
