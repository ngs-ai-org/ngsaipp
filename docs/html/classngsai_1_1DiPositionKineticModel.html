ngsai::DiPositionKineticModel Class Reference

The DiPositionKineticModel class is made to model the raw IPD and PWD signal from PacBio CCS. This model models a window of IPD/PWD signal of L consecutive bp. L is named the model size. This model is a special case of the PairWiseKineticModel. The PairWiseKineticModel accounted for all N = ((L*L) - L) / 2 pair-wise position interaction whereas this one only considers L-1 interactions between any two directly neighbouring positions in the window. Except for this, everything is the same as in PairWiseKineticModel. More...

Detailed Description

The DiPositionKineticModel class is made to model the raw IPD and PWD signal from PacBio CCS. This model models a window of IPD/PWD signal of L consecutive bp. L is named the model size. This model is a special case of the PairWiseKineticModel. The PairWiseKineticModel accounted for all N = ((L*L) - L) / 2 pair-wise position interaction whereas this one only considers L-1 interactions between any two directly neighbouring positions in the window. Except for this, everything is the same as in PairWiseKineticModel.
