Implementation Ruzzo and Tompa 1999's algorithm (A Linear Time ALgorithm for Finding all Maximal Scoring Subsequences). Finds all maximum positive scoring segments within the given sequence of scores.

Parameters
scoresthe sequence of positive and negative scores. Positive scores means good, negative scores means bad.
Returns
a list of maximum scoring segments. Each pair in the list indicates a range [from,to) in the score sequence that delimitates a maximum scoring segment. Each range can be retrieved using pair.first and pair.second respectively.

Implementation of Cs≈©ros 2004's MinLength-Cover's algorithm (Maximum Scoring Segment Sets). Finds a set of maximum positive scoring segments within the given sequence of scores given the constrains. This set of segments (k-cover) has the optimal score given the constrains.

Parameters
thesequence of positive and negative scores. Positive scores means good, negative scores means bad.
alphathe segment set size penaly. The penalized segment set score is ≈µ(S) = w(S) - a*|S| where w(S) is the raw segment set score and |S| is the number of segments in the set.
min_length_lowthe minimum size of a low scoring segment.
min_length_highthe minimum size of a high scoring segment.
Returns
a list of maximum scoring segments. Each pair in the list indicates a range [from,to) in the score sequence that delimitates a maximum scoring segment. Each range can be retrieved using pair.first and pair.second respectively.
